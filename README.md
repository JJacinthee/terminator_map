# 🌍 全球晨昏线 · 昼夜时长

**在线访问** → [点击打开](https://JJacinthee.github.io/terminator-map/)

实时可视化全球任意地点的昼夜状态与时长，适合旅游做攻略、天文爱好者、教学演示使用。

## 功能

- 等矩形投影世界地图，含经纬网格
- 实时晨昏线（橙色），根据日期与 UTC 时刻精确计算
- 夜侧阴影（天文公式，误差几分钟以内）
- 太阳直射点实时位置（黄色圆点）
- 月份 / 日期 / UTC 时刻三个滑块控制
- 内置约 200 个全球主要城市，支持中英文搜索
- 点击地图查看任意地点的日出、日落、昼长、全年变化曲线
- 支持手机端访问

## 技术

纯前端，无后端依赖。使用 [D3.js](https://d3js.org/) 渲染地图与晨昏线计算。

## 使用说明

直接在浏览器打开 `index.html` 即可，或访问在线版本。
